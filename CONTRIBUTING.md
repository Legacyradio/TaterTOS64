# Contributing to TaterTOS64v3

TaterTOS64v3 is a proprietary project and is not open to unsolicited code
contributions.

## Do Not Submit Unsolicited Code

- Do not open pull requests with code changes unless you have prior written
  approval from the project owner.
- Do not assume that publishing this repository implies permission to reuse,
  extend, or redistribute the code.
- Do not submit third-party code for inclusion unless ownership and licensing
  have been cleared in writing first.

## Issues and Discussions

- Bug reports and high-level discussion may be useful.
- Feature proposals, collaboration requests, and licensing inquiries should be
  framed as discussion first, not as code submissions.

## Contribution Terms

If the project owner explicitly requests or approves a contribution, any
submission is made with the understanding that:

- no obligation exists to merge, review, or respond;
- no rights are granted back to the contributor except as stated in a separate
  written agreement;
- ownership, licensing, and any commercial rights must be resolved in writing
  before the contribution is accepted.

If you want to discuss a commercial arrangement, source license, evaluation
access, or approved collaboration path, see [COMMERCIAL.md](COMMERCIAL.md).
