# Contributing to TaterTOS64v3

TaterTOS64v3 is open source under the GPL v3. Contributions are welcome.

## How to Contribute

- Bug reports and feature proposals via GitHub Issues.
- Pull requests for code changes — please keep them focused and testable.
- All contributions must be compatible with the GPL v3 license.

## Contribution Terms

By submitting a contribution, you agree that your work is licensed under the
same GPL v3 license as the rest of the project.
