# TaterTOS64v3

TaterTOS64v3 is a free and open-source 64-bit operating system developed from scratch.

## Status

- Current milestone: the system boots and is alive on target hardware.
- Repository focus: active development, debugging history, and internal build
  assets for the project owner.
