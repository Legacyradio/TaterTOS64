Summary
- Fixed SMP trampoline GDT: use 32-bit code/data for protected-mode entry, 64-bit code for long-mode jump.

Change
1) src/drivers/smp/trampoline.asm
   - GDT now has 32-bit code (0x08) + 32-bit data (0x10) + 64-bit code (0x18).
   - Long-mode far jump uses selector 0x18.

Why
- APs were #GP looping at 0x8008 in protected mode because code segment had L=1.
  PM entry must use a 32-bit code segment (L=0, D=1).
