CODE CHANGES

1) src/kernel/fs/vfs.c
   - Added `#include "ntfs.h"` for NTFS API/structs.
   - Added `SEC_FS_NTFS` secondary filesystem type.
   - Added `sec_ntfs_ops` and read-only stub handlers:
     `ntfs_write_stub`, `ntfs_create_stub`, `ntfs_mkdir_stub`, `ntfs_unlink_stub`.
   - Wired `sec_ntfs_ops` in `sec_ops_init` to NTFS vfs ops + read-only stubs.
   - Added `mount_secondary_ntfs()` to mount NTFS partitions read-only.
   - `mount_secondary_candidate()` now supports NTFS candidates.
   - Secondary GPT scan now accepts NTFS Basic Data GUID + `ntfs_probe()`.
   - Added `boot_media_seen` guard: if a FAT32 partition scores >0 for
     TaterTOS user-space files, skip secondary mounts entirely to avoid
     touching live boot media.
   - Added `ntfs_idx` selection: prefer NTFS as primary secondary mount
     when FAT32 score is 0 (EFI-style FAT32), otherwise keep ToTFS > FAT32.

No changes to ramdisk primary or vfs_init; NTFS only participates in
`vfs_mount_secondary()` (live-media secondary mount path).
