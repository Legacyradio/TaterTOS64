MAC Context Setup (iwl_mac_ctx_add):
  - Sends MAC_CONTEXT_CMD (0x28) to create STA-mode MAC context
  - Sets our MAC address, TSF ID, filter flags

PHY Context Setup (iwl_phy_ctx_add):
  - Sends PHY_CONTEXT_CMD (0x08) to configure radio
  - Channel number, band (2.4/5 GHz), 20 MHz width, TX/RX chain masks (A+B)

Binding (iwl_binding_add):
  - Sends BINDING_CONTEXT_CMD (0x2B) to bind MAC context to PHY context
  - Required before any TX/RX can happen

UMAC Scan (iwl_start_scan + iwl_collect_scan_results):
  - Sends SCAN_REQ_UMAC (0x0C, group 0x0D) with:
    - 11 channels (2.4 GHz, channels 1-11)
    - Active dwell 30ms, passive dwell 110ms per channel
    - Wildcard probe request template (broadcast SSID)
  - Probe request builder includes: SSID IE, Supported Rates (1-18 Mbps),
    Extended Rates (24-54 Mbps)
  - Collects results for up to 5 seconds
  - Parses beacon/probe response frames: SSID, BSSID, channel (DS params),
    capability info, beacon interval, RSN IE detection
  - Deduplicates by BSSID
  - Prints each found BSS with SSID, channel, BSSID, [WPA2]/[OPEN] tag

TX Management Frames (iwl_tx_mgmt):
  - Wraps 802.11 frame in TX_CMD (0x1C) with:
    - 1 Mbps CCK rate (basic rate for management frames)
    - ACK expected, 3 retries, infinite lifetime
  - Sends via host command interface

Authentication (iwl_send_auth):
  - Builds Open System Authentication frame:
    - FC = MGMT | AUTH subtype
    - DA/BSSID = target AP, SA = our MAC
    - Auth body: algorithm=OPEN, seq=1, status=0
  - Sends via TX_CMD

Association (iwl_send_assoc_req):
  - Builds Association Request frame:
    - FC = MGMT | ASSOC_REQ subtype
    - Capability: ESS, short preamble, short slot
    - Listen interval = 10
    - SSID IE, Supported Rates, Extended Rates

Response Waiting (iwl_wait_mgmt_resp):
  - Polls RX queue for auth/assoc response frames
  - Checks frame control subtype match
  - Parses status code from response body
  - For assoc: also extracts AID

Full Association Flow (iwl_associate):
  1. Update PHY context to target channel
  2. Send auth → wait for auth response (1s timeout)
  3. Send assoc request → wait for assoc response (1s timeout)

Init Sequence (iwl_mac_init):
  1. Store MAC address
  2. Add MAC context (STA mode)
  3. Add PHY context (channel 1, 2.4 GHz)
  4. Create binding
