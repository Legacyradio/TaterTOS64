SESSION CHANGES (all from fry777 continuation):

1. E1000 NIC DRIVER — src/drivers/net/e1000.c (NEW, ~300 LOC)
   - Full Intel 82540EM driver for QEMU (-nic user,model=e1000)
   - PCI probe (8086:100E/100C/100F), MMIO BAR0, EEPROM MAC read
   - 32-entry TX/RX descriptor rings, 8KB DMA buffers
   - Polled mode (no IRQ), legacy descriptors
   - Hooked into netcore.c (TX fallback + RX poll + attach)

2. NETCORE UPDATES — src/drivers/net/netcore.c
   - Added e1000 driver hooks (is_ready, tx_packet, rx_poll, set_rx_callback)
   - TX path: i219 → e1000 → wifi fallback
   - net_poll: i219 → e1000 → wifi
   - netcore_attach_e1000() copies MAC + registers RX callback

3. GUI: DYNAMIC START MENU — src/user/gui/gui.c
   - Removed hardcoded app list from discover_apps()

   - draw_desktop_icons(): pixel blit + centered text label
   - desktop_icon_hit(): click detection → launch_app_index()
