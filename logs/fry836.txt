DIFFS APPLIED IN fry836
=====================================================================

A. EXPANDED <string> shim — basic_string<CharT, Traits, Allocator> template
   - Converted std::string from a single class to a template
     parameterised on CharT/Traits/Allocator.
   - Added: char_traits<>, allocator_default<>, npos, size_type,
     iterator/const_iterator (CharT*), front/back, max_size, resize
     (single-arg + two-arg), reserve, clear, [] operator.
   - Standard aliases: string, u8string, u16string, u32string, wstring.
   - Reason: fmt and simdutf reference std::basic_string<...> in
     formatter declarations; bare class wasn't enough.

B. EXPANDED <string_view> shim — basic_string_view<CharT, Traits> template
   - Same pattern: turned the class into a template parameterised
     on CharT.
   - Aliases: string_view, u16string_view, u32string_view, wstring_view.

C. EXPANDED <system_error> shim — full error_code/error_category/
   system_error class.
   - Added: error_category virtual base, generic_category() and
     system_category() singletons, error_code class (value + cat
     pointer + message()/operator bool()/comparison), system_error
     subclass of runtime_error, is_error_code_enum<errc>,
     make_error_code(errc).
   - Reason: fmt declares format_system_error returning
     std::system_error.

D. EXPANDED <type_traits> shim — many additions
   - decay / decay_t / remove_extent / add_pointer / is_function
     (forward + full)
   - is_fundamental / is_null_pointer / is_scalar / is_object /
     is_empty / is_polymorphic / is_abstract
   - is_constructible / is_default_constructible /
     is_copy_constructible / is_move_constructible
   - is_base_of / is_convertible (SFINAE-based, GCC-portable)
   - is_assignable / is_copy_assignable / is_move_assignable
   - is_trivially_copy_constructible / is_trivially_move_constructible
   - is_nothrow_constructible / is_nothrow_default_constructible /
     is_nothrow_copy_constructible / is_nothrow_move_constructible /
     is_nothrow_assignable / is_nothrow_copy_assignable /
     is_nothrow_move_assignable
   - is_member_pointer / is_member_function_pointer /
     is_member_object_pointer
   - declval forward-declared at top using a private
     remove_reference_fwd (so SFINAE helpers can use it before the
     canonical remove_reference appears).

E. EXPANDED <memory> shim — full allocator_traits surface
   - allocator_traits<Alloc> with: value_type, pointer/const_pointer,
     size_type/difference_type, propagate_on_container_copy/move/swap,
     is_always_equal (all false / true singletons),
     allocate/deallocate/max_size/select_on_container_copy_construction,
     construct/destroy.

F. EXPANDED <algorithm> shim
   - copy_backward (used by fast_float bigint).
   - Removed swap (it's in <utility>; was double-defined).

G. PATCHED math.h
   - Replaced macro form of isfinite/isinf/isnan/isnormal/signbit
     with C++ inline overloads OUTSIDE the extern "C" block (C still
     gets the macro form). Reason: `std::isfinite(x)` macro-expanded
     to `std::__builtin_isfinite(x)` which is invalid in C++.

H. EXPANDED <cmath> shim
   - Added inline forwarders for isfinite/isinf/isnan/isnormal so
     `std::isfinite(...)` etc. compile.

I. PATCHED <climits> shim
   - Added LLONG_MIN/LLONG_MAX/ULLONG_MAX (gcc freestanding limits.h
     omits the C99 long-long values).

J. PATCHED <limits> shim
   - Fixed the typo `_TATERTOS_NL_INT_AGAIN` (left over from fry834).
     Replaced with proper hand-written specialisations for char16_t
     and char32_t.
   - The `_TATERTOS_NL_FLOAT(long double, ...)` macro invocation
     also choked because `long double` is two tokens; replaced with
     a hand-written specialisation for long double.

K. MAKEFILE
   - AK_CXXFLAGS gained `-DFMT_USE_LOCALE=0 -DFMT_USE_FCNTL=0`.
     These tell fmt to bypass <locale> and <fcntl.h> in its
     internal-formatter paths — neither is available in our shim
     and neither is needed for AK's actual fmt usage.

L. POSIX shim additions (3 small)
   - <time.h>: clockid_t typedef, timegm() inline forwarder,
     extern timezone/daylight/tzname[2], CLOCK_MONOTONIC_COARSE +
     CLOCK_REALTIME_COARSE aliases.
