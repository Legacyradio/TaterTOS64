CARGO CONFIG
=====================================================================
src/user/apps/ladybird-rust-vendor/.cargo/config.toml:

  [source.crates-io]
  replace-with = "vendored-sources"

  [source.vendored-sources]
  directory = "/home/.../ladybird-rust-vendor/vendor"

  [build]
  target = "x86_64-unknown-tatertos"

  [target.x86_64-unknown-tatertos]
  linker = "/usr/bin/x86_64-elf-ld"

This file makes cargo:
  1. Refuse to fetch from crates.io (offline-strict).
  2. Default to our custom target on `cargo build`.
  3. Use TaterTOS's cross-linker for any final-binary linking.

(Cargo's stable target-name resolution still wants the JSON
path on the command line — that's a -Zunstable-options command
we run from the Makefile when we get to Phase R-7. The config
above provides the linker default and the offline source.)
