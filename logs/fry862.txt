=====================================================================
Create a functional smoke test for Ladybird's LibCore networking
abstractions on TaterTOS. This test verifies that the event loop,
socket creation, and host resolution all work together to perform
a real network request.

=====================================================================

- Uses Core::EventLoop for asynchronous notification handling.
- Uses Core::TCPSocket and Core::BufferedTCPSocket for high-level
  stream management.
- Includes a 10-second watchdog timer to prevent hangs.

- Linked with LibCore.a, AK.a, and the TaterTOS LibC.

underlying networking stack (from kernel up to LibCore) is solid.
This test follows the "no shortcuts" rule by using the actual
high-level abstractions that the browser will use.
