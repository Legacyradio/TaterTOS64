CONFIRMED AUDIT FINDINGS
=====================================================================
1. link_smoke.cpp is not a real engine smoke test.
   It includes LibJS/Runtime/VM.h and LibWeb/Bindings/Intrinsics.h,
   then returns 0. This proves only archive linkage, not LibGC, JS VM,
   DOM creation, HTML parsing, layout, painting, or event integration.

2. LibGfx TaterTOS backend contains shortcuts.
   PainterTaterTOS ignores transforms, clipping, source rectangles,
   scaling, global alpha, filters, compositing, and all path rendering.
   Bitmap alpha conversion is a TaterTOS no-op stub.

3. LibRegex contains hidden functional gaps.
   lre_check_stack_overflow() always returns no overflow, named capture
   group extraction is skipped, unicode_sets is not mapped, and
   is_single_non_bmp_literal() is stubbed false.

4. TaterTOS input integration drops events.
   TaterTOSInputNotifier drains keyboard and mouse syscalls and discards
   the events instead of posting real Core input events.

5. TaterSurf removal is approved by Zack.
   Do not restore tatersurf into Makefile/build_iso.sh. If preservation
   is needed, keep only a local/reference copy outside the active build
   path. Current active work should continue without TaterSurf.

6. GUI regression requires separate repair.
   src/user/gui/gui.c was reduced from the full TaterWin GUI manager to
   a placeholder loop. This needs restoration in a dedicated step, not
   buried inside the Ladybird link-smoke work.
