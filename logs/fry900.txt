Kernel handlers implemented in src/kernel/proc/syscall.c:

SYS_UNAME(132):
- Writes the 5x65 struct utsname fields: TaterTOS, taterbox, 1.0.0,
  TaterTOS64v3 #1 SMP, x86_64.

SYS_SYSINFO(133):
- Computes uptime from HPET counter.
- Reports total/free RAM via pmm_get_total_pages() / pmm_get_used_pages().
- Counts active processes from the proc[] table.
- Builds a 64-byte buffer manually (no memset/size_t in freestanding kernel).

SYS_GETRUSAGE(134):
- Returns a zeroed 144-byte rusage struct.
- Only supports RUSAGE_SELF (0).

SYS_GETPRIORITY(135):
- Returns 0 (nice=0). Validates which == PRIO_PROCESS(0).

SYS_SETPRIORITY(136):
- No-op, returns success.

SYS_FSYNC(137) / SYS_FDATASYNC(138):
- Ramdisk is always synced. Validates fd range, returns success.

SYS_SCHED_GETAFFINITY(139):
- Queries smp_cpu_count() for online CPUs.
- Builds bitmask byte array (up to 128 bytes).
- Returns the copy size as POSIX requires.

SYS_SCHED_SETAFFINITY(140):
- No-op, returns success.
