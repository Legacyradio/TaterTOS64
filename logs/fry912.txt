=== TOOLS (mktotfs.c, totcopy.c) ===

mktotfs: writes superblock/bitmaps/inodes/root dir, uses fopen(file, "r+b")
totcopy: reads/writes existing ToTFS image entries, same fopen pattern
Both work on any fopen-accessible path including /dev/sda on Linux.
Real limitation: they are host-side build tools, not TaterTOS-native
installer apps that can run from within the OS.
