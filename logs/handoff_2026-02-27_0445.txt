=== CURRENT STATE ===
Foundation for a modern Windowing System is laid. We moved from a static, hardcoded debug GUI to a dynamic, memory-backed desktop environment.

Major Breakthroughs:
1. Dynamic Memory (Userland Heap):
   - Implemented SYS_SBRK (syscall 22).
   - Added malloc/free/calloc/realloc to LibC.
   - User processes can now allocate memory dynamically (heap starts at 1TB mark).

2. Shared Memory (SHM):
   - Implemented SYS_SHM_ALLOC (23) and SYS_SHM_MAP (24).
   - Foundation for high-performance buffer passing between apps and the compositor.

3. libgfx & 8x16 Font:
   - Created a standalone graphics library.
   - Replaced 5x7 calculator font with professional 8x16 bitmap font.
   - Added vertical gradient support and double buffering.

4. TaterTOS64v3 Premium GUI:
   - Completely rewritten gui.c.
   - Modern "VS-Code" dark theme (0x181818).
   - Floating windows with shadows and accented title bars.
   - Sidebar with application launch tracking.
   - Smooth arrow cursor (hardware-independent).
