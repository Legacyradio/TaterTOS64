SESSION SUMMARY
---------------
Fixed three critical AML parser bugs that caused the namespace tree to be
incomplete on all hardware. Added If/Else/While block handling. Added
namespace node count to on-screen diagnostics.

PROBLEM
-------
ACPI diagnostics on Dell Precision 7530 showed:
  EC: FAIL, Batt found=0, BL found=0
Both battery and backlight discovery use ns_walk() — if BOTH are found=0,
the namespace tree itself must be incomplete. Investigation found:

ROOT CAUSE: OpRegion/Mutex/Event parsers assumed PkgLength where the ACPI
spec defines none. Each OpRegion (very common — dozens per DSDT) caused
parse_block to skip forward by a garbage length (hundreds to thousands of
bytes), obliterating subsequent Device/Method/Name declarations. Additional
issue: If/Else/While blocks (which may contain Device declarations) were
not handled, causing devices inside conditionals to be invisible.

FIXES APPLIED
-------------
1) parse_opregion(): Removed bogus parse_pkg_length(). OpRegion per ACPI
   spec is: NameString + RegionSpace(byte) + Offset(TermArg) + Len(TermArg).
   No PkgLength. Now parses flat and returns end position.

2) parse_mutex(): Removed bogus parse_pkg_length(). Mutex per ACPI spec is:
   NameString + SyncFlags(byte). No PkgLength.

3) parse_event(): Removed bogus parse_pkg_length(). Event per ACPI spec is:
   NameString only. No PkgLength.

4) parse_block() for OpRegion/Mutex/Event: Changed from reading phantom
   PkgLength to using return value from parser function (p = parse_xxx()).

5) parse_block() added If (0xA0), Else (0xA1), While (0xA2): These DO
   have PkgLength. Reads PkgLength, recursively calls parse_block on body
   to discover nested Device/Method/Name declarations.

6) namespace.c: Added ns_node_count() — walks tree and counts nodes.

7) extended.h + extended.c + libc.h + sysinfo.c: Added ns_nodes field to
   ACPI diagnostics struct and display. Shows on-screen how many namespace
   nodes were parsed.

SCOPE: These are ACPI spec compliance fixes. Affect ALL hardware, not
Dell-specific. Any machine with OpRegion/Mutex/Event/If blocks in its
DSDT/SSDTs was previously getting an incomplete namespace.
